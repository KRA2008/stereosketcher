StereoSketcher

StereoSketcher is a stereoscopic 3D drawing toy. You can watch this tutorial video on how to operate StereoSketcher, or see the chart below for the controls.

KEYS

KeyFunction
1Cross eye view mode
2Parallel eye view mode
3Red/Cyan view mode
SpacebarHide dots/switch to display-only mode
qmove selected shapes to back layer
wmove selected shapes one layer backward
emove selected shapes one layer forward
rmove selected shapes to front layer
ashift dots into screen
sshift dots out of screen
dcreate line between two selected dots
fcreate face between three selected dots
zmake selected lines thinner
xmake selected lines thicker
cmove images apart
vmove images together

MOUSE

ButtonGestureModifierTargetFunction
leftclickopen spacecreate dot
leftclickshiftopen spacecreate dot and select it (cumulative select)
leftclickitemtoggle select item and deselect all others
leftclickshiftitemtoggle select item (cumulative select)
leftclick and dragselected dotmove selected dots
leftclick and draganywhereselect dots contained in rectangle
rightclickanywheredeselect all
rightclick and draganywherepan
wheelrollanywherezoom

Pro tips

    • To crop your image before uploading, just resize your browser window.
      I did some test drawings along the way.
      If somebody sends me a link to some of their sketches I'll link to them here too.
      This helps.

    • Other help

    • Some colors work better than others in red/cyan mode. i.e. red doesn't work well.
      If you are getting the images to line up in parallel or cross viewing, but the image is weird or even 3D in the wrong direction, you might be using the wrong view mode - try switching view modes.
      This was tested in Firefox during development, results in other browsers may vary.
