StereoSketcher

StereoSketcher is a 3D stereoscopic image drawing toy. There are 3 modes the program can use to display the drawings.

  1. Cross-eye
    • Tutorial 1
    • Tutorial 2
  2. Magic eye
    • Tutorial 1
    • Tutorial 2
  3. Red/Cyan
    • Free 3D glasses
    • Cheap 3D glasses

KEYS

KeyFunction
1Cross-eye view mode
2Magic eye view mode
3Red/Cyan view mode
SpacebarHide dots/switch to display-only mode
qmove selected shapes to back
wmove selected shapes one layer backward
emove selected shapes one layer forward
rmove selected shapes to front
ashift dots into screen
sshift dots out of screen
dcreate line between two selected dots
fcreate face between three selected dots
zmake selected lines thinner
xmake selected lines thicker
cmove images apart
vmove images together
