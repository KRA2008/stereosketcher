StereoSketcher

StereoSketcher is a 3D stereoscopic image drawing toy. There are 3 modes the program can use to display the drawings.

  1. Cross Eye
    • Tutorial
  2. Parallel Eye
    • Tutorial 1
  3. Red/Cyan
    • Free 3D glasses
    • Cheap 3D glasses

KEYS

KeyFunction
1Cross eye view mode
2Parallel eye view mode
3Red/Cyan view mode
SpacebarHide dots/switch to display-only mode
qmove selected shapes to back
wmove selected shapes one layer backward
emove selected shapes one layer forward
rmove selected shapes to front
ashift dots into screen
sshift dots out of screen
dcreate line between two selected dots
fcreate face between three selected dots
zmake selected lines thinner
xmake selected lines thicker
cmove images apart
vmove images together

FAQ

There haven't been any Q's A'd yet, but you can ask here or send me an email: StereoSketcher \at/ StereoSketcher \dot/ com. Any commentary is appreciated!

Some tips

    • To crop your image before uploading, just resize your browser window.
      If you are getting the images to line up in parallel or cross viewing, but the image is 3D in the wrong direction, you might accidentally be using the wrong view mode - try switching view modes.
